# React Agent

A Python framework for building agent systems with reasoning (ReAct pattern).

## Features

- ReAct (Reasoning + Acting) agent framework
- Tool registry and management
- Multiple LLM support (OpenAI, etc.)
- Memory management
- Verbose logging and debugging

## Installation

```bash
pip install react-agent
```
or
```bash
pip install git+https://github.com/yourname/react-agent.git
```

## Quick Start

```python
from react_agent.core import ReactAgent

agent = ReactAgent()
# Use the agent...
```
